## Overview

The accumulator module provides functionality for accumulating streaming data into HEALPix cells with incremental statistics, enabling efficient processing of large datasets in a streaming or batch context.

## Core Classes

### StreamingStats

Container for streaming statistics using Welford's algorithm.

```python
class StreamingStats:
    def __init__(self)
    def update(self, values)
    def merge(self, other: 'StreamingStats')
    
    @property
    def mean(self) -> float
    
    @property
    def std(self) -> float
    
    def to_dict(self) -> Dict[str, Any]
    
    @classmethod
    def from_dict(cls, d: Dict[str, Any]) -> 'StreamingStats'
```

**Attributes:**
- `n`: Number of observations
- `sum`: Sum of values
- `sum_sq`: Sum of squared values
- `min_val`: Minimum value
- `max_val`: Maximum value

**Methods:**

#### update

Add new observations to the running statistics.

<ParamField path="values" type="scalar | list | np.ndarray" required>
  New observations (automatically converted to 1D float64 array)
</ParamField>

**Example:**
```python
from healpyxel.accumulator import StreamingStats
import numpy as np

stats = StreamingStats()
stats.update(1.0)              # Scalar
stats.update([2.0, 3.0])       # List
stats.update(np.array([4.0]))  # Array

print(f"Count: {stats.n}")
print(f"Mean: {stats.mean}")
print(f"Std: {stats.std}")
```

#### merge

Merge with another StreamingStats object (for parallel processing).

<ParamField path="other" type="StreamingStats" required>
  Another StreamingStats instance to merge
</ParamField>

**Example:**
```python
stats1 = StreamingStats()
stats1.update([1, 2, 3])

stats2 = StreamingStats()
stats2.update([4, 5, 6])

stats1.merge(stats2)
print(f"Combined mean: {stats1.mean}")  # Mean of [1,2,3,4,5,6]
```

### CellAccumulator

Accumulator for a single HEALPix cell.

```python
class CellAccumulator:
    def __init__(self, use_tdigest: bool = True)
    def update(self, column: str, values: np.ndarray)
    def merge(self, other: 'CellAccumulator')
    def to_dict(self) -> dict
    
    @classmethod
    def from_dict(cls, d: Dict[str, Any], use_tdigest: bool = True) -> 'CellAccumulator'
```

**Attributes:**
- `stats_by_column`: Dict mapping column names to StreamingStats
- `tdigests`: Dict mapping column names to T-Digest (if enabled)
- `use_tdigest`: Whether T-Digest is enabled for approximate percentiles

**Methods:**

#### update

Add observations for a specific column.

<ParamField path="column" type="str" required>
  Column name
</ParamField>

<ParamField path="values" type="np.ndarray" required>
  Array of new observations
</ParamField>

**Example:**
```python
from healpyxel.accumulator import CellAccumulator
import numpy as np

acc = CellAccumulator(use_tdigest=True)
acc.update('reflectance', np.array([0.5, 0.6, 0.55]))
acc.update('temperature', np.array([250, 255, 252]))

print(f"Reflectance stats: n={acc.stats_by_column['reflectance'].n}")
print(f"Mean reflectance: {acc.stats_by_column['reflectance'].mean}")
```

#### merge

Merge with another CellAccumulator (for parallel processing).

<ParamField path="other" type="CellAccumulator" required>
  Another CellAccumulator instance to merge
</ParamField>

## Core Functions

### accumulate_batch

Process one batch of data and update accumulator state.

```python
def accumulate_batch(
    new_data: pd.DataFrame,
    sidecar: pd.DataFrame,
    value_columns: List[str],
    existing_state: Optional[Dict[int, CellAccumulator]] = None,
    use_tdigest: bool = True,
    filter_expr: Optional[str] = None
) -> Dict[int, CellAccumulator]
```

<ParamField path="new_data" type="pd.DataFrame" required>
  DataFrame with observations (index = source_id)
</ParamField>

<ParamField path="sidecar" type="pd.DataFrame" required>
  HEALPix mapping with columns ['source_id', 'healpix_id']
</ParamField>

<ParamField path="value_columns" type="List[str]" required>
  Columns to accumulate
</ParamField>

<ParamField path="existing_state" type="Dict[int, CellAccumulator]" default="None">
  Previous accumulator state (None for first batch)
</ParamField>

<ParamField path="use_tdigest" type="bool" default="True">
  Enable T-Digest for approximate percentiles
</ParamField>

<ParamField path="filter_expr" type="str" default="None">
  Optional pandas query expression to filter data
</ParamField>

<ResponseField name="return" type="Dict[int, CellAccumulator]">
  Updated state dictionary mapping healpix_id to CellAccumulator
</ResponseField>

**Note:**
In fuzzy mode, sidecar has multiple rows per source_id (one per touched HEALPix cell). This function handles the many-to-one relationship correctly by grouping on healpix_id.

**Example:**
```python
import pandas as pd
from healpyxel.accumulator import accumulate_batch

# First batch
data1 = pd.read_parquet('day001.parquet')
sidecar1 = pd.read_parquet('day001_sidecar.parquet')

state = accumulate_batch(
    new_data=data1,
    sidecar=sidecar1,
    value_columns=['r750', 'r950'],
    existing_state=None
)

print(f"Cells after batch 1: {len(state)}")

# Second batch (incremental)
data2 = pd.read_parquet('day002.parquet')
sidecar2 = pd.read_parquet('day002_sidecar.parquet')

state = accumulate_batch(
    new_data=data2,
    sidecar=sidecar2,
    value_columns=['r750', 'r950'],
    existing_state=state
)

print(f"Cells after batch 2: {len(state)}")
```

### save_state

Serialize accumulated state dict to Parquet with metadata.

```python
def save_state(
    state,
    output_path,
    meta,
    processing_metadata=None
)
```

<ParamField path="state" type="Dict[int, CellAccumulator]" required>
  State dictionary from accumulate_batch()
</ParamField>

<ParamField path="output_path" type="Path" required>
  Path to write parquet file
</ParamField>

<ParamField path="meta" type="HEALPyxelxMetadata" required>
  HEALPyxelxMetadata instance (provides nside, order, lon_convention)
</ParamField>

<ParamField path="processing_metadata" type="dict" default="None">
  Optional dict with batch info
</ParamField>

**Example:**
```python
from pathlib import Path
from healpyxel.accumulator import save_state
from healpyxel.metadata import HEALPyxelxMetadata

meta = HEALPyxelxMetadata(
    nside=512,
    order='nested',
    mode='fuzzy',
    lon_convention='0_360'
)

save_state(
    state=state,
    output_path=Path('state_v001.parquet'),
    meta=meta,
    processing_metadata={'batch_id': 'day001', 'n_cells': len(state)}
)
```

### load_state

Load accumulator state and HEALPix metadata from parquet file.

```python
def load_state(
    input_path: Path,
    use_tdigest: bool = True
) -> Tuple[Dict[int, CellAccumulator], Optional[HEALPyxelxMetadata]]
```

<ParamField path="input_path" type="Path" required>
  Path to state parquet file
</ParamField>

<ParamField path="use_tdigest" type="bool" default="True">
  Whether to restore T-Digest data
</ParamField>

<ResponseField name="return" type="Tuple">
  Tuple of (state dict, metadata) where metadata may be None if not found
</ResponseField>

**Raises:**
- `FileNotFoundError`: If state file does not exist

**Example:**
```python
from pathlib import Path
from healpyxel.accumulator import load_state

state, meta = load_state(Path('state_v001.parquet'))
print(f"Loaded {len(state)} cells")
print(f"nside={meta.nside}, mode={meta.mode}")
```

## Validation Functions

### validate_accumulator_sidecar_compatibility

Validate that accumulator state is compatible with sidecar file.

```python
def validate_accumulator_sidecar_compatibility(
    state_meta: HEALPyxelxMetadata,
    sidecar_meta: HEALPyxelxMetadata
) -> dict
```

<ParamField path="state_meta" type="HEALPyxelxMetadata" required>
  Metadata from loaded state file
</ParamField>

<ParamField path="sidecar_meta" type="HEALPyxelxMetadata" required>
  Metadata from sidecar file
</ParamField>

<ResponseField name="return" type="dict">
  Dict with validation results: `valid`, `errors`, `warnings`
</ResponseField>

**Raises:**
- `AssertionError`: If critical parameters mismatch (nside, mode, order)

**Checks:**
- **Critical**: nside match (must be identical)
- **Critical**: mode match (must be identical)
- **Critical**: order match (must be identical)
- **Warning**: lon_convention (different values generate warning)

**Example:**
```python
from healpyxel.accumulator import (
    load_state,
    validate_accumulator_sidecar_compatibility
)
from healpyxel.metadata import HEALPyxelxMetadata

# Load state metadata
state, state_meta = load_state(Path('state_v001.parquet'))

# Load sidecar metadata
sidecar_meta = HEALPyxelxMetadata.from_parquet(Path('sidecar.parquet'))

# Validate compatibility
try:
    result = validate_accumulator_sidecar_compatibility(state_meta, sidecar_meta)
    print("✓ State and sidecar are compatible")
except AssertionError as e:
    print(f"❌ Incompatible: {e}")
```

### find_sidecar

Attempt to find matching sidecar file for input data.

```python
def find_sidecar(
    input_path: Path,
    nside: Optional[int] = None,
    mode: str = 'fuzzy'
) -> Optional[Path]
```

<ParamField path="input_path" type="Path" required>
  Path to input parquet file
</ParamField>

<ParamField path="nside" type="int" default="None">
  Desired nside (if None, finds any matching sidecar)
</ParamField>

<ParamField path="mode" type="str" default="'fuzzy'">
  Assignment mode ('fuzzy' or 'strict')
</ParamField>

<ResponseField name="return" type="Optional[Path]">
  Path to matching sidecar file, or None if not found
</ResponseField>

**Search Locations:**
1. Same directory as input file
2. `sidecars/` subdirectory

**Pattern:**
```
{base_name}.cell-healpix_assignment-{mode}_nside-{nside}_order-nested.parquet
```

**Example:**
```python
from pathlib import Path
from healpyxel.accumulator import find_sidecar

sidecar = find_sidecar(
    Path('observations.parquet'),
    nside=512,
    mode='fuzzy'
)

if sidecar:
    print(f"Found sidecar: {sidecar}")
else:
    print("No matching sidecar found")
```

## CLI Entry Point

### main

Main entry point for accumulator command.

```python
def main(argv=None) -> int
```

**Command-line Usage:**

```bash
# Initialize state from first batch
healpyxel_accumulator \
  --input day001.parquet \
  --sidecar day001_nside512.parquet \
  --columns r750 r950 vis_slope \
  --state-output state_v001.parquet

# Incremental update with second batch
healpyxel_accumulator \
  --input day002.parquet \
  --sidecar day002_nside512.parquet \
  --columns r750 r950 vis_slope \
  --state-input state_v001.parquet \
  --state-output state_v002.parquet

# With data filtering
healpyxel_accumulator \
  --input day003.parquet \
  --sidecar day003_nside512.parquet \
  --columns r750 r950 vis_slope \
  --state-input state_v002.parquet \
  --state-output state_v003.parquet \
  --filter "quality > 0.5 and solar_zenith < 80"
```

**Key Arguments:**
- `--input`: Input parquet file with new observations
- `--sidecar`: HEALPix sidecar mapping file (auto-detected if not specified)
- `--columns`: Column names to accumulate
- `--state-input`: Existing state file for incremental update
- `--state-output`: Output state file path
- `--filter`: Pandas query expression to filter data
- `--no-tdigest`: Disable T-Digest percentile tracking
- `--batch-id`: Batch identifier for metadata tracking
