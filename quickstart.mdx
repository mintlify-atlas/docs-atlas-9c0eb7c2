## Installation

First, install HEALPyxel:

<CodeGroup>
```bash pip
pip install healpyxel
```

```bash conda
conda install -c conda-forge healpy
pip install healpyxel
```
</CodeGroup>

<Tip>
For geospatial operations, install the geospatial extras: `pip install healpyxel[geospatial]`
</Tip>

## Basic Workflow

HEALPyxel follows a simple **Split-Apply-Combine** pattern:

<Steps>
  <Step title="Split: Create Sidecar">
    Map observations to HEALPix cells
    
    ```python
    from healpyxel import sidecar
    import geopandas as gpd
    
    # Load your geospatial data
    gdf = gpd.read_parquet('observations.parquet')
    
    # Generate sidecar mapping
    sidecar_df = sidecar.generate(
        gdf,
        nside=64,
        mode="fuzzy",
        order="nested"
    )
    ```
  </Step>
  
  <Step title="Apply: Aggregate Data">
    Calculate statistics per HEALPix cell
    
    ```python
    from healpyxel import aggregate
    
    # Compute statistics
    agg_df = aggregate.by_sidecar(
        original=gdf,
        sidecar=sidecar_df,
        value_columns=["reflectance", "temperature"],
        aggs=["median", "robust_std"],
        min_count=3
    )
    
    print(agg_df.head())
    ```
  </Step>
  
  <Step title="Combine: Visualize">
    Add geometry and visualize results
    
    ```python
    from healpyxel.geospatial import healpix_to_geodataframe
    
    # Attach HEALPix cell geometries
    cells_gdf = healpix_to_geodataframe(
        nside=64,
        pixels=agg_df["healpix_id"].values
    )
    
    result_gdf = cells_gdf.merge(agg_df, on="healpix_id")
    
    # Visualize with geopandas
    result_gdf.plot(column="reflectance_median", cmap="viridis")
    ```
  </Step>
</Steps>

## CLI Workflow

Prefer command-line tools? Use the CLI:

<Steps>
  <Step title="Generate Sidecar">
    ```bash
    healpyxel_sidecar \
      --input observations.parquet \
      --nside 64 \
      --mode fuzzy \
      --output-dir output/
    ```
  </Step>
  
  <Step title="Aggregate Statistics">
    ```bash
    healpyxel_aggregate \
      --input observations.parquet \
      --sidecar-dir output/ \
      --columns reflectance temperature \
      --aggs median robust_std \
      --min-count 3
    ```
  </Step>
  
  <Step title="Add Geometry">
    ```bash
    healpyxel_to_geoparquet \
      --aggregate-path output/*-aggregated.parquet \
      --output-dir output/
    ```
  </Step>
</Steps>

## Streaming Workflow

For large datasets that don't fit in memory:

```python
from healpyxel import accumulator, finalize

# Initialize state
state = None

# Process batches incrementally
for batch_file in batch_files:
    batch_df = pd.read_parquet(batch_file)
    sidecar_df = sidecar.generate(batch_df, nside=64)
    
    state = accumulator.update_state(
        batch=batch_df,
        sidecar=sidecar_df,
        value_columns=["reflectance"],
        state=state
    )

# Finalize statistics
final_df = finalize.from_state(
    state=state,
    aggs=["mean", "std", "median"]
)
```
